ReasonManip：激励大模型多模态推理以直接操控机器人

INCENTIVIZING MULTIMODAL REASONING IN LARGE MODELS FOR DIRECT ROBOT MANIPULATION

将 LLM / LMM 与机械臂底层控制衔接的前沿思路：不显式训练传统 Action Head 或黑盒端到端 VLA，而是让模型以 System-2 式慢推理（算术、几何与逻辑）在自然语言或可解析结构中给出下一步 6D 位姿等量。下文按动机、方法与实验整理。

一、动机与痛点（Motivation）

  • VLA 的「快思考」局限：典型视觉–语言–动作模型（如 RT-1、OpenVLA）多属 System-1：图像直接映到动作，深层推理弱，OOD 场景易失效。
  • 欧拉角 / 四元数对 LLM 不友好：直接输出姿态若用大量 \(\sin/\cos\) 与角域缠绕，模型易产生数值幻觉与符号错误。
  • 系统级能力：希望模型在「出手」前能进行碰撞预判、失败分析与自纠错等，而不仅是单步回归。

二、核心方法（Methodology）

1. 轴向空间表示（Axis-based Spatial Representation）

放弃以欧拉角为主的三维旋转参数化，改用三个正交单位向量描述朝向；物体与机械臂位姿在统一框架下编码。常见记法包括位置 \((x,y,z)\)、外廓尺寸 \((l,w,h)\)，以及三组轴（示例记号与论文一致时采用原文）：

  • \((x_a,y_a,z_a)\)：纵向轴（如物体最长方向或夹爪指向）；
  • \((x_b,y_b,z_b)\)：副法向轴（如夹爪开合方向）；
  • \((x_n,y_n,z_n)\)：法向轴（如贴合面法向）。

动机（与预训练对齐）：LLM 语料中向量加减、叉乘等比三角恒等更「自然」。例如法向可由 \(\mathbf{a} \times \mathbf{b}\)（再归一化）得到，便于链式推理，减少纯三角函数链路上的错误。

注：若将上述标量全部展开，维度为 \(3{+}3{+}3{\times}3\)；组会材料中「12 维」以论文最终定义为准。

2. 第一阶段：小规模高质量 SFT

  • 数据：以开源多模态大模型 Qwen2.5-VL-72B 在仿真中、在人类短提示（如「继续」「调角度」）下试错，构造约 65 条多轮推理对话，而非数十万条示教轨迹。
  • 微调：用该数据监督微调较小模型（如 7B），学习 CoT 格式：场景分析 → 碰撞与可行性 → 向量几何 → 输出下一步位姿。

3. 第二阶段：GRPO 强化学习（Group Relative Policy Optimization）

在仿真中对同一任务并行采样 \(K\) 条不同推理–控制轨迹；稀疏奖励：仅当任务成功时 \(r=1\)，否则 \(0\)。在组内根据 \(K\) 条结果的统计构造相对优势 \(\hat{A}_k\)，驱使策略偏向能导向成功的推理与数值推导。

目标在精神上接近「组内归一优势 + 带裁剪的策略比率」，与 PPO 同属信赖域族；示意写法（时间步 / token 级求和与系数以原文为准）：

$$ \mathcal{L}_{\mathrm{GRPO}}(\theta) = \mathbb{E}\left[ \frac{1}{K}\sum_{k=1}^{K} \min\!\left( \rho_k\,\hat{A}_k,\; \mathrm{clip}(\rho_k,\,1-\epsilon,\,1+\epsilon)\,\hat{A}_k \right) \right] $$

其中 \(\rho_k\) 为第 \(k\) 条轨迹上新旧策略概率比（或论文采用的等价形式），\(\hat{A}_k\) 由组内回报相对基线（如组均值、方差）得到。旧策略采样、对新策略优化的套路与 PPO 笔记中的裁剪思想可对照阅读。

效果直觉：将死板模仿扩展为在成功/失败信号下探索何种几何推理链更可靠，强化与物理不变量（如法向对齐）一致的行为，而非死记像素统计。

三、实验与结果（Experiments）

  • 数据效率（SIMPLER，7 项任务）：仅 65 条 SFT + GRPO，平均成功率约 45.4%；对比 OpenVLA 约 14.0%（其训练规模通常远大于此）。
  • OOD 零样本（MetaWorld）：在 SIMPLER 训练后无微调迁移到物理与渲染均不同的 MetaWorld；RT-1-X、OpenVLA 等报告为 0%，而 ReasonManip（SFT+GRPO）在抓取类任务上约 96.7%。
  • 视角鲁棒性：更换相机视角后成功率仍高（如 View 1 约 96.7% → View 2 达 100%），与「学几何常识而非单一视角纹理」的解释一致。

四、Takeaway

表示层与 LLM 预训练能力对齐（向量与叉乘优于硬写欧拉角）；学习层用小而精的 SFT 定型 CoT，再用 GRPO 用稀疏成功信号放大可靠推理。与纯端到端 VLA 形成互补视角。末端位姿仍难毫米级精确时，可与 被动柔顺末端（如折纸抓手）等硬件思路一起考虑鲁棒性。
